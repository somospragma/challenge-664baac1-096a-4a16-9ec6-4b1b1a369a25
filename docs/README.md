# Proyecto de Versionamiento de Código con Git

Este proyecto demuestra los fundamentos de versionamiento de código utilizando Git. Incluye la configuración inicial del repositorio, trabajo en ramas, resolución de conflictos y uso de comandos avanzados.

## Instalación
1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/tu-repo.git
   ```
2. Entra en el directorio del proyecto:
   ```bash
   cd tu-repo
   ```

## Ejecución
1. Ejecuta el archivo principal:
   ```bash
   python src/main.py
   ```

## Pruebas
1. Ejecuta las pruebas unitarias:
   ```bash
   python -m unittest tests/test_main.py
   ```
